# 血圧手帳 💊 BP & Period Log

血圧・脈拍の記録・グラフ・通院管理・生理周期管理ができるPWAアプリです。

## 主な機能
- 🩺 血圧・脈拍の記録（朝/夜）・ダブル錠服薬マーク
- 📅 カレンダー表示（通院日・生理周期・血圧カテゴリ色）
- 🏥 通院管理（次回通院予測・期間内最高血圧サマリ）
- 📈 グラフ（7/14/21/28/60/90日・測定値ラベル・基準ライン）
- 🌼 生理期間帯グラフ表示
- ⚙️ 目標値・警告値・処方週数の設定
- 🗂️ JSONエクスポート/インポートでデータ管理
- 📱 PWA対応（ホーム画面に追加可能・オフライン動作）

## GitHub Pages デプロイ手順

1. このリポジトリをGitHubにプッシュ（全ファイル）
2. Settings → Pages → Source を `main` ブランチの `/(root)` に設定
3. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開

## ホーム画面への追加（PWAインストール）

- **iPhone/iPad**: Safariで開く → 共有ボタン → 「ホーム画面に追加」
- **Android**: Chromeで開く → メニュー → 「ホーム画面に追加」

## データについて

- データはブラウザのlocalStorageに保存されます
- 「データ」タブからJSONファイルでエクスポート/インポート可能
- 定期的なバックアップを推奨します
